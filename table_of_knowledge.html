Tables of knowledge

Table for HTML code

Code Description
<table> The code which allows you to start developing tables.
<thead> (table head) This sets up your title row of your table.
<tbody> (table body) This sets up the body of your table.
<tfoot> (table footer) This sets up the footer of your table.
<tr> (table row) This sets up a row in your table.
<td> (table data) This allows you to enter data into the table.
<td colspan="x"> (colum span) This allows your data to move across "x" colums in the same row.
<td rowspan="x"> (row span) This allows your data to move across "x" rows in the same colum.
The basic code needed to make a table in html.
